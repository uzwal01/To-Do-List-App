# To-Do-List-App

📝 To-Do List App

A simple and intuitive To-Do List Application built with React.js, designed to help you manage your daily tasks efficiently. The app supports adding, completing, and deleting tasks, with data persistence using localStorage.

🚀 Features
1. Add New Tasks: Quickly add tasks to your list using the input field.
2. Mark Tasks as Complete: Toggle between completed and incomplete states for each task.
3. Delete Tasks: Remove tasks from your list with a single click.
4. Data Persistence: Tasks are saved in the browser's localStorage, ensuring data remains even after refreshing or closing the page.
5. Clean UI: A minimal and responsive design with modern styles.

🛠️ Tech Stack
1. React.js: Frontend framework for building the app interface.
2. CSS/Tailwind CSS: For styling and responsive design.
3. LocalStorage: For data persistence.
